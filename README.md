# lego-dateradio 日期选择

### 参数说明：
- typeValue: 0, //类型值   0为长期，1为自定义
- startValue: '',  开始日期
- endValue: '',     结束日期
- minDate: '',  最小日期
- maxDate: '',  最大日期
- format: 'YYYY-MM-DD HH:mm:ss',
- onChange(){}
